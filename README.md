## How I work

- **Report from source, never hand-typed.** Every number traces back to data.
- **Human-in-the-loop.** Automation drafts and proposes; people approve and send.
- **Tested and reproducible.** Pipelines and tooling ship with tests and one-command runs.
- **Privacy first.** Data classification and masking are built in, not bolted on.
